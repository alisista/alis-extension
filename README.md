## Chrome Extension

chrome://extensions/

## Prerequisites

+ node

## Setup

```
$ git clone https://github.com/alisista/alis-extension
$ cd alis-extension
$ npm i
$ npm run start
```

## Local Install

After execute `npm run start` , add `app` directory to chrome://extensions/

## Deploy

```
$ npm run deploy
```
